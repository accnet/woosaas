│   │   ├── [siteId]/               # Site-level pages
│   │   │   ├── layout.tsx          # Pass-through
│   │   │   ├── overview/           # Performance Summary
│   │   │   ├── trend/              # Trend Analysis

## 1. Portfolio Dashboard (`/dashboard`)

### Hiện tại
- Hai section: **Workspace Summary** (metric cards + runbook) và **Site Portfolio** (site cards)
- Metric cards: Total Sites, Active, Verified, Pending
- Runbook: 4 bước hướng dẫn cơ bản
- Site cards hiển thị tên, domain, trạng thái, last signal, 3 action buttons

### Vấn đề
1. **Thiếu insight tổng quan thực sự**:
   - Không có tổng hợp doanh thu / traffic từ tất cả sites
   - Workspace summary chỉ đếm số lượng sites, không có KPI kinh doanh
   - Không có biểu đồ xu hướng tổng hợp
2. **Runbook không cập nhật theo context**:
   - Chỉ là nội dung tĩnh, không dựa vào trạng thái thực tế của user
   - Không suggested next actions dựa trên site nào đang cần attention nhất
3. **Thiếu "cảnh báo sớm"**:
   - Không hiển thị sites có vấn đề (health degraded, không nhận event lâu)
   - Không có notification về pipeline issues

### Đề xuất
```
a) Thêm Workspace KPI bar:
   - Tổng pageviews / sessions / revenue / orders từ tất cả sites
   - So sánh period-over-period
b) Thêm "Needs Attention" section:
   - Sites with degraded health
   - Sites không nhận event > 24h
c) Smart runbook:
   - Dynamic dựa trên trạng thái thực tế
```

## 12. Vấn Đề Kỹ Thuật

### State Management
- Auth: Zustand ✅
- Navigation: URL-based ✅
- Date range: Local state ❌ → Context
- Site list: Fetch lại mỗi page ❌ → Cache

### Data Fetching
- Pattern: useEffect + useState ❌ → React Query

### Performance
- Thiếu debounce cho search filters ❌
- Site list fetched riêng ❌
- localStorage không async

### Accessibility
- Thiếu aria-labels ⚠️
- Table sorting không có aria-sort ⚠️
