**Date:** 2026-05-08
**Scope:** Full dashboard interface analysis, including navigation, pages, components, and data visualization.

## 1. Trang Dashboard Chính (`/dashboard`)

### Phân tích

Trang Portfolio Dashboard đã được cấu trúc tương đối tốt với:
- `Workspace Summary` — thống kê tổng quan (Total Sites, Active, Verified, Pending)
- `Runbook` — gợi ý hành động theo trạng thái
- `Site Portfolio` — danh sách site với tracking state, last signal, và CTA buttons

### Cải tiến đã thực hiện

| Tính năng | Mô tả |
|-----------|-------|
| **Needs Attention** | Section mới hiển thị site pending > 24h hoặc verified nhưng không có event trong 7 ngày. Có `border-l-amber-400` để trực quan hóa vấn đề. |
| **Smart Runbook** | Logic động: hiển thị hướng dẫn setup khi chưa có site, cảnh báo/resolve khi có issue, trạng thái healthy khi mọi thứ ổn |
| **InfoCell component** | Component nhỏ gọn để hiển thị label + value theo dạng card (Status detail, Last signal, Next step) |
| **AlertTriangle section** | Header có icon cảnh báo amber, filter sites cần chú ý |

### Gợi ý thêm

- Thêm `EmptyState` cho Needs Attention khi không có vấn đề
- Hiển thị realtime online users count ở phần Runbook

## 2. Navigation & Dashboard Shell

### Phân tích

Dashboard shell có 3-layer navigation: App Rail (icon trái), Site Sidebar (menu trái rộng 300px), Top Nav (header). Đầy đủ site switcher với search, pin, recent sites.

### Vấn đề phát hiện

1. **Environment section trong Sidebar** — hiển thị localhost:8080/3000 cứng nhắc, không có giá trị production
2. **Thiếu Quick Actions** — không có shortcuts đến realtime, health check, funnel từ sidebar

### Cải tiến đã thực hiện

| Thay đổi | Chi tiết |
|----------|----------|
| **Replace Environment → Quick Actions** | Chuyển section hiển thị API/Dashboard URLs thành các link nhanh đến realtime, pipeline health, conversion funnel |
| **Thêm Recommendations section** | Trong SiteSidebar: hiển thị "Finish onboarding" nếu chưa active, "Watch realtime" nếu đã active |

### Gợi ý thêm

- Site switcher có thể hiển thị trạng thái tracking (chấm màu) ngay trong dropdown
- Thêm keyboard shortcut (`Cmd+K`) cho site search

## 3. DataTable Component

### Phân tích

Component `DataTable` mới được xây dựng với TypeScript generics, hỗ trợ sort, column alignment, custom render.

### Vấn đề phát hiện

- Không có TypeScript constraint có thể gây lỗi render
- Thiếu type safety cho `sortValue` khi cast `Record<string, unknown>`

### Cải tiến đã thực hiện

| Thay đổi | Chi tiết |
|----------|----------|
| **Loại bỏ Record constraint** | `DataTable<T extends Record<string, unknown>>` → `DataTable<T>` cho phép bất kỳ type nào |
| **Sort an toàn** | Sử dụng `(a as Record<string, unknown>)[sortKey]` chỉ trong trường hợp không có `sortValue` |
| **Add emptyTitle/emptyBody** | Props cho phép custom empty state message |
| **Thêm multi-column sắp xếp** | Sort indicator (`ArrowUpDown`, `ArrowUp`, `ArrowDown`) với ARIA sort attributes |

## 7. Tổng quan các vấn đề còn lại

### High Priority

1. **TypeScript build lỗi** — Cần chạy `next build` kiểm tra
2. **DateRangeContext chưa được dùng** — Các page hiện tại dùng state local, chưa tận dụng context
3. **Thiếu keyboard navigation** — Site switcher có keyboard, nhưng DataTable chưa có keyboard sort

### Medium Priority

1. **EmptyState consistency** — Một số page dùng EmptyState component, số khác dùng inline
2. **Loading skeleton** — Chỉ có LoadingSpinner, chưa có skeleton loading cho tables/cards
3. **Error boundaries** — Chưa có ErrorBoundary wrapper component-level
4. **Accessibility** — Thiếu `aria-labels` trên icon buttons, thiếu focus management cho drawer

### Low Priority

1. **Animation** — Thiếu transition cho table sort, chart update
2. **Theming** — Một số color codes cứng (`#6366f1`, `#22c55e`)
3. **Responsive** — 2xl grid columns có thể break trên màn hình nhỏ
